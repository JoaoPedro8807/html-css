Fontes

As vezes algum cliente quer uma fonte específica que apenas ele tenha, para uma palavra se tornar mais original, então nao podemos usar o google fonts nesse caso. Existe varios sites para pegar fontes externas, mas nesse exemplo usaremos o DaFonte, mas o cliente ja pode dar um arquivo de Fonte .TTF OU .OTF Tem navegadores que funcionam melhores com TTF outros com OTF, mas podemos colocar os dois, nesse caso eu só consegui baixar font com TTF, então coloquei 2 fontes diferentes apenas como exemplo, mas o certo é colocar 2 fontes igual, mas de formatos diferentes (.TTF, e .OTF)
