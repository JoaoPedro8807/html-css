Estilos Externos

Capítulo 1

    Os estilos externos é o mais usado, pois ele consegue editar todas as paginas do site com um unico programa, no caso do estilos local
    isso não seria possivel, por isso é mais viavel utilizar o externo.
    
    
    Para criar o estilo externo, criamos um link na área de HEAD, e selecionamos LINK:css, ele irá criar um target para um novo arquivo 
    css, apenas o target será criando, o arquivo ainda não foi criado. Para criar o arquivo rapidamente, aperte CTRL e clique em cima do href='', o vsstudio
    vai falar que não existe o arquivo e vai sugerir criar automaticamente, dai é so clicar em criar


    Aí dentro do arquivo css, você pode começar a editar o CSS, lembre-se sempre de criar a regra charset  no começo do arquivo 
    basta digitar '@charset' e colocar  'UTF-8' para ele reconhecer as asentuações
